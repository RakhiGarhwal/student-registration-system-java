# Student Registration System using Java, JDBC & Tomcat

A web-based Student Registration System built using **Java**, **JDBC**, **Servlet/JSP**, and **MySQL**.  
The system allows students to register, log in, and view their details through a simple dashboard.  
It demonstrates database connectivity and basic CRUD operations using JDBC.
